## Public support

Open a GitHub issue for reproducible bugs, region requests, documentation
errors, and accessibility feedback.

Never post private timing documents, personal details, access tokens, full
Windows usernames, or sensitive logs in public.

## Religious and timing questions

Sakoon cannot issue religious rulings or declare a calculated time official.
For authoritative guidance, use the relevant awqaf publication and ask the
local mosque or qualified authority.
