# Sakoon v1.0.0

The first public release of **Sakoon · سكون** for Windows.

Sakoon mutes system audio before each configured azaan window, keeps the
computer quiet through the window, and restores the previous audio state
afterwards.

## Highlights

- Official monthly CSV/XLSX timing tables take priority.
- Imported tables are versioned and checksummed.
- Calculated solar times are local, optional fallback values and are always
  labelled **CALCULATED — UNOFFICIAL**.
- Verify view compares official and calculated times to expose suspicious
  import shifts.
- Audio is muted at the endpoint; the volume level is never moved.
- Previous mute and volume state are captured before audio changes.
- Recovery state helps prevent a crash from leaving the PC silent.
- Missed events are not intentionally replayed after sleep.
- Friday Dhuhr supports a configurable Jumu'ah window.
- Tray snooze supports meetings.
- No account, advertisements, or telemetry.
- No administrator access required for normal use.

## Included files

- `Sakoon.exe` — Windows application
- `Sakoon-Table-Format-Example-NOT-OFFICIAL.csv` — synthetic format example
- `SHA256SUMS.txt` — release-asset checksums

## Install

1. Download `Sakoon.exe` and `SHA256SUMS.txt`.
2. Verify the executable's SHA-256 hash.
3. Run the executable.
4. Select the correct region.
5. Upload the current official authority table when available.
6. Review the import preview before activation.

## Important timing notice

Sakoon is not a religious authority. The relevant awqaf publication and local
mosque always take precedence. Calculated values are unofficial and Sakoon
should not be the only prayer reminder.

## Known limitations

- Windows only.
- New unsigned executables may trigger a Windows reputation warning.
- Spreadsheet layouts with decorative or merged cells may require a cleaned
  copy.
- Audio behaviour can vary across Bluetooth, docks, vendor drivers, virtual
  audio devices, and endpoint changes.
- Hardware-specific sleep and hibernation behaviour may require additional
  testing.

## Privacy

Sakoon operates entirely on the local machine. It does not require an account, ads, or
telemetry for its core operation.
