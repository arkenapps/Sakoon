The mall moment

A small courtesy, made automatic.

In many GCC malls, music stops when the azaan begins. Sakoon brings that same moment of stillness to the Windows desktop.

“The building itself observes the moment. Sakoon lets your computer do the same.”

You may be working with music playing, watching a video, or using an application when the azaan begins. Instead of reaching for the volume key five times a day, Sakoon handles the audio window for you and hands it straight back.

It does not play the azaan and it does not replace a prayer reminder. It simply manages your PC audio around the timing source you have selected.

The name means stillness. That is the entire idea: mute, pause, restore, and get out of the way.

Timing philosophy

Official authority first. Astronomy as a watchdog.

Prayer times vary by authority, region, method, and deliberate precaution offsets. Sakoon keeps those two responsibilities separate rather than blending them.

PREFERRED

Official uploaded tables

The monthly CSV or Excel table published by the relevant authority is treated as the source of truth.

  • Versioned on every upload — nothing is overwritten
  • Checksummed, with visible provenance
  • Strict parsing; a suspicious file is rejected, not guessed at
  • Arabic column headers recognised
CALCULATED — UNOFFICIAL

Local astronomical fallback

Solar calculation runs entirely on the machine and is always labelled unofficial, because it cannot reproduce an authority's precaution offsets.

  • Used only when no official table is present
  • Powers the Verify view's official-versus-calculated delta
  • A small, stable delta is healthy; a sudden jump means a bad import
  • Never presented as authoritative

Built for real Windows behaviour

Audio is treated as a recovery problem.

The failure that matters most for an audio utility is leaving a computer unexpectedly silent. Five rules exist to prevent it.

01

Mute, never re-level

The output device is muted; your volume stays where you set it.

02

Capture, never toggle

The previous mute and volume state is preserved.

03

Recovery first

Restore information is written before audio is changed.

04

No stale replay

A missed window is not intentionally replayed after wake.

05

The user always wins

Manual unmute or override causes Sakoon to stand down.

The first time you run it, Windows will warn you

Sakoon is not code-signed — a signing certificate is an annual cost this project has not taken on. Windows therefore shows “Windows protected your PC” for any application it has not seen before. That warning is about reputation, not about anything detected in the file.

To continue: click More info, then Run anyway. The prompt stops appearing once Windows has seen the application a few times.

If you would rather verify before trusting it: every release publishes SHA256SUMS.txt. Compare it against Get-FileHash Sakoon.exe in PowerShell, and only ever download from the official arkenapps/Sakoon release page.

Questions

Clear answers, without pretending software is infallible.

Is Sakoon a religious authority?

No. Sakoon is a convenience utility. The relevant awqaf authority and local mosque always take precedence, and Sakoon should never be the only prayer reminder.

Does Sakoon fetch prayer times from the internet?

Its preferred source is an official monthly CSV or Excel table uploaded by you. Its astronomical fallback runs locally and is clearly identified as unofficial.

Does it open a browser or a local web server?

No. From version 2.0.0 the interface renders in its own application window and Sakoon opens no network ports at all. The window uses Microsoft Edge WebView2, which ships with Windows 11 and current Windows 10; on the rare machine without it, Windows offers a one-time free install on first launch.

What happens when I close the window?

Sakoon returns to the system tray and keeps guarding your audio. Use Quit in the tray menu to exit completely, which also restores your audio on the way out.

Why does Windows say the app is unrecognised?

Because it is unsigned and new. SmartScreen flags anything without an established reputation. Choose More info, then Run anyway, and verify the published SHA-256 hash if you want certainty about the file you downloaded.

Can it leave my PC permanently muted?

Recovery state is written before audio is changed, and orphaned state is restored on the next start. Hardware and drivers can still fail, so reproducible restoration problems should be reported.

Does it need administrator access?

No. Normal use and the visible Windows Startup shortcut both stay inside your own user profile.

Is the source code public?

No. The GitHub repository is public for documentation, issue tracking, release notes, and official binary distribution. The Go source code is not published.

Important timing notice

Sakoon is not a religious authority and cannot certify a prayer time. Official awqaf publications and your local mosque override every imported, displayed, or calculated value. Software can fail; never rely on Sakoon as your only prayer reminder.
