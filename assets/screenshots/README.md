# Screenshot replacement guide

Before each public release, replace the four SVG placeholders with final,
sanitised screenshots while preserving the filenames or updating README links.

Recommended capture rules:

- use demonstration timing data rather than a private authority file;
- remove usernames, file paths, device names, and machine names;
- keep one consistent Windows scaling level;
- capture the Night theme for the primary dashboard;
- show the UNOFFICIAL badge clearly in the verification example;
- do not show source-code folders or private build paths.
